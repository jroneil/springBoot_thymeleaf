This version of the application features a glassmorphic UI, JPA persistence, and a modular layout system.

About This Project

This is a Spring Boot 3.5.0 demo showcasing reusable Thymeleaf fragment layouts without the Thymeleaf Layout Dialect. Every page uses the strict layout :: layout(content, pageTitle) contract.

The Roles module demonstrates full CRUD operations backed by an H2 in-memory database, Spring Data JPA, and Bean Validation with inline error messages.
